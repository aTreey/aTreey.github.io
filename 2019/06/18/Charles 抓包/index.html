原理

简单理解如下：

  • 正常请求

    客户端 ——> 服务器

  • Charles 抓包

    客户端 ——> 代理 （Charles）——> 服务器

启动Charles

安装Charles证书

如果安装失败，请检查以前是否已经安装过，如果已经安装过并且已经失效，请删除失效证书后再安装

打开钥匙串–> 选择Charles CA -> 双击证书 –> 信任证书

设置 https

- Proxy -> SSL Proxying Settings -> SSL Proxying -> Add
- Host: * 为需要过滤的域名地址,
- *: 表示不过滤Port, 固定为443 `*`表示任意端口

真机抓包
